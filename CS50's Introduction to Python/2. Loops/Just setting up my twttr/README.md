# Twttr

A small Python program that shortens text by removing vowels.

The program asks the user to enter a string of text. It then outputs the same text, but with all vowels removed. Both uppercase and lowercase vowels are omitted.

## Task

Implement a program in `twttr.py` that:

* prompts the user for a string of text
* removes all vowels from the input
* treats uppercase and lowercase vowels the same
* keeps all consonants, numbers, spaces, and punctuation
* outputs the shortened version of the text

## Example

Input:

```text
Twitter
```

Output:

```text
Twttr
```

Input:

```text
Hello, world!
```

Output:

```text
Hll, wrld!
```

Input:

```text
CS50
```

Output:

```text
CS50
```
