# Math Interpreter

A small Python program that works as a simple arithmetic interpreter.

The program asks the user to enter a mathematical expression in the format `x y z`, where `x` and `z` are integers and `y` is an arithmetic operator. It then calculates the result and outputs it as a floating-point number formatted to one decimal place.

## Task

Implement a program in `interpreter.py` that:

* prompts the user for an arithmetic expression
* expects input in the format `x y z`
* supports addition with `+`
* supports subtraction with `-`
* supports multiplication with `*`
* supports division with `/`
* converts the result to a floating-point value
* formats the result to one decimal place
