# Einstein

A small Python program that calculates energy using Einstein’s mass-energy equivalence formula.

The program asks the user to enter mass as an integer in kilograms. It then calculates the equivalent amount of energy in Joules using the formula `E = mc²`, where `c` is the speed of light.

## Task

Implement a program in `einstein.py` that:

* prompts the user for mass in kilograms
* expects the mass to be entered as an integer
* uses the speed of light as `300000000` meters per second
* calculates energy using the formula `E = mc²`
* outputs the equivalent energy in Joules
* displays the result as an integer

## Formula

```text id="k7m4qa"
E = m * c²
```

Where:

```text id="x9p2vn"
m = mass in kilograms
c = 300000000 meters per second
```

Input:

```text id="q6z8lm"
1
```
