# Indoor Voice

A small Python program that converts user input to lowercase.

The program asks the user to enter a string of text. It then outputs the same text entirely in lowercase, while leaving punctuation, numbers, and whitespace unchanged.

## Task

Implement a program in `indoor.py` that:

* prompts the user for input
* converts all uppercase letters to lowercase
* leaves punctuation unchanged
* leaves whitespace unchanged
* outputs the converted text

## Example

Input:

```text id="x8k4lm"
HELLO, WORLD!
```

Output:

```text id="q2v9ra"
hello, world!
```

Input:

```text id="m6d3zp"
CS50 Python
```

Output:

```text id="r7n5cw"
cs50 python
```

Input:

```text id="k9t2yb"
This is my indoor voice.
```

Output:

```text id="p4x8sn"
this is my indoor voice.
```
