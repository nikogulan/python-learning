# Playback Speed

A small Python program that slows down text by replacing spaces with three dots.

The program asks the user to enter a string of text. It then outputs the same text, but every space is replaced with `...`, creating a slower, paused effect between words.

## Task

Implement a program in `playback.py` that:

* prompts the user for input
* keeps the original text unchanged except for spaces
* replaces each space with `...`
* outputs the modified text

## Example

Input:

```text id="h7k3qm"
This is CS50
```

Output:

```text id="v9p2la"
This...is...CS50
```

Input:

```text id="m4x8zn"
Hello world
```

Output:

```text id="q6d1rp"
Hello...world
```

Input:

```text id="n8c5yb"
I speak too fast
```

Output:

```text id="s2w7kt"
I...speak...too...fast
```
